Aim

Objective of the experiment

Experiment Objective

The aim of this experiment is to understand the General method for solving Linear Differential Equations and to analyze their solutions using systematic steps.

By the end of this experiment, students will be able to:

  • Understand the basic concept and form of linear differential equations.
  • Apply the general method to solve different types of linear differential equations.
  • Interpret the solution and understand its physical or mathematical meaning.
  • Improve problem-solving skills through step-by-step analysis of equations.

Theory

Background concepts & definitions

What is General Method?

The General Method is a technique used to find the particular integral (P.I.) of a linear differential equation with constant coefficients.

It is based on expressing the differential equation in operator form:

φ(D)y = f(x)

where the method used to find the particular integral depends on the nature of φ(D).

General Solution of LDE

It is the solution of D.E. in which No. of arbitary constants is equal to order of differential equation.

G. S. = Complimentary function (C.F.) + Particular Integral (P.I.)

y = yc + yp

Complimentary function (C.F)

Complement function is a solution of [φ(D)y] = 0 i.e. f(x)= 0

Step 1: Find roots of the polynomial φ(D) = 0,where the polynomial equation φ(D)= 0 is known as an auxiliary equation.

Step 2: Depending upon nature of the roots of the auxiliary equation, complement function is as follows:

Sr. No. Roots of the Auxiliary Equation C.F.
1 If m1, m2, m3, ...., mn are real and distinct roots y = c1em1x + c2em2x + c3em3x + .... + cnemnx
2 If m1, m2, m3, ...., mn are real and repeated roots

No. of arbitrary constants is equal to Number of roots 		a) If roots are m1, m1, m1, m2, m2, m3, m4, m5
y = (c1x2 + c2x + c3) em1x + (c4x + c5) em2x + c6em3x + c7em4x + c8em5x

b) If roots are m1, m2, m3, m3, m3, m4, m5, m6, m6
y = c1em1x + c2em2x + (c3x2 + c4x + c5) em3x + c6em4x + c7em5x + (c8x + c9) em6x
3 If roots are complex α ± iβ, m yc = eαx (c1 cos βx + c2 sin βx) + c3emx
4 If complex roots are repeated α ± iβ, α ± iβ, α ± iβ, m yc = eαx [(c1x2 + c2x + c3) cos βx + (c4x2 + c5x + c6) sin βx] + c7emx

Example:

d2y/dx2 - 6 dy/dx + 9y = 0

1. Write the Auxiliary Equation (A.E.):

Replace d/dx with D or m:

m2 - 6m + 9 = 0

2. Find the Roots:

(m - 3)(m - 3) = 0

Roots are m1 = 3 and m2 = 3 (Real and Equal).

3. Write the Complementary Function (yc):

Instead of C1e3x + C2e3x (which would just simplify to one term), we use:

yc = (C1 + C2x)e3x

Particular Integral

The particular integral of L.D.E. φ D y = f(x) is given by

yp = 1 φ(D) f(x)

Sr. No. yp = 1 / φ(D) f(x) Formula
1 yp = 1 / (D − m) f(x) yp = emx ∫ e−mx f(x) dx
2 yp = 1 / (D + m) f(x) yp = e−mx ∫ emx f(x) dx
3 yp = 1 / [(D − m1)(D − m2)] f(x) yp = 1/(D − m1) [1/(D − m2) f(x)]

OR

yp = 1/(D − m2) [1/(D − m1) f(x)]
Using Partial Fraction yp = 1/(m1 − m2) [1/(D − m1) − 1/(D − m2)] f(x)
4 yp = 1/D f(x) , yp = 1/D² f(x) yp = ∫ f(x) dx , yp = ∫∫ f(x) dx

Solve

d²y/dx² + dy/dx = 1/(1+ex)

Solution:

Let D = d/dx

(D² + D)y = 1/(1+ex)

Auxiliary equation:

D² + D = 0

D(D + 1) = 0

D = 0 , -1

Complementary Function:

yc = c₁ + c₂e-x

Particular Integral:

yp = [1/D(D+1)] × 1/(1+ex)

Using partial fractions:

yp = [1/D − 1/(D+1)] × 1/(1+ex)

yp = ∫ dx/(1+ex) − e-x ∫ exdx/(1+ex)

For first integral: e-x + 1 = t

For second integral: 1 + ex = u

yp = −ln(e-x + 1) − e-xln(1 + ex)

Final Answer:

y = c₁ + c₂e-x − ln(e-x + 1) − e-xln(1 + ex)
